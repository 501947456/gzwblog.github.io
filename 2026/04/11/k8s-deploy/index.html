目录结构

1. 环境要求

1.1 集群要求

  • Kubernetes 1.22+
  • 至少 3 个可用节点
  • 每个节点至少 4 核 CPU / 8GB 内存
  • 支持持久卷 (PersistentVolume)

1.2 资源要求

Elasticsearch 2 核 4Gi 80Gi

8. 安全建议

  1. 修改默认密码: 生产环境务必修改 Secret 中的密码
  2. 启用 SSL: 启用 HTTP SSL 加密
  3. 网络策略: 配置 NetworkPolicy 限制访问
  4. RBAC: 使用 Kibana Spaces 和角色管理
  5. 定期备份: 配置快照和恢复策略
  6. 监控告警: 配置 Watcher 进行异常告警
  7. 更新版本: 定期更新到最新稳定版本
